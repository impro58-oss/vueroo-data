## 🔒 SECURITY MEASURES IN PLACE

### **1. Skill Vetting Process** (Now Required)

**Before installing ANY skill, I will:**
- ✅ Read the full SKILL.md file
- ✅ Check for suspicious code, URLs, or instructions
- ✅ Verify the source (official registry vs. unknown)
- ✅ Look for hidden commands or obfuscated scripts
- ✅ Check metadata for required permissions
- ✅ Get YOUR approval before installation

**Red flags to watch for:**
- Requests to delete files or data
- Instructions to modify system files
- Commands to send data to external servers
- Obfuscated or encoded scripts
- Requests for sensitive credentials
- Self-modification instructions

- ❌ Send your personal data to external services

- ❌ Share conversation logs without permission
- ❌ Reveal system configuration details publicly

**What I WILL do:**

- ✅ Use official APIs only

**I will NOT:**
