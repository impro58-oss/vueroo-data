Health Check

NEW

Monitor system health, temperature, disk space, and OpenClaw status. Proactive alerting for issues.

Features

  • Gateway status monitoring
  • CPU, memory, disk usage tracking
  • Temperature monitoring
  • Process health checks
  • Telegram alerts on critical issues

Usage

# Run full system health check & "C:\Users\impro\.openclaw\workspace\skills\healthcheck\healthcheck.ps1" -Command "run" # Check specific component & "...\healthcheck.ps1" -Command "check" -Component "gateway"

Scheduled

✅ Runs automatically every 30 minutes via cron job.

PDF Tools

NEW

Extract text, tables, and data from PDF documents. Process medical papers, research, and reports.

Features

  • Text extraction
  • Table parsing to CSV
  • Batch processing
  • OCR support (with Tesseract)
  • Medical paper optimization

Usage

# Extract text from PDF pdf-tools extract document.pdf --output text.txt # Extract tables to CSV pdf-tools tables document.pdf --output tables.csv # Extract specific pages pdf-tools extract document.pdf --pages 1-5 --output summary.txt

Advanced Cron

NEW

Enhanced scheduling with retry logic, dependencies, error handling, and monitoring.

Features

  • Retry logic (3 attempts)
  • Job dependencies
  • Error handling
  • Notifications
  • Health dashboard

Usage

# Create advanced cron job cron-advanced add --name "Crypto Scan" --command "python scan.py" --schedule "0 */4 * * *" --retry 3 # Add job with dependencies cron-advanced add --name "Report" --command "generate.py" --after "Crypto Scan" # Monitor job status cron-advanced status
