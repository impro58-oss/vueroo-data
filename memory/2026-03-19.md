User reported Portfolio Matrix dropdowns not working - filters not responding, data not displaying. Root cause identified as browser security blocking local file loading (`file://` protocol prevents loading external JSON/JS files).

### Investigation Process
1. **Diagnosed iframe security issue** - Browsers block iframes from loading local files
2. **Identified Chart.js failures** - Breaking JavaScript execution and preventing filters
3. **Discovered data loading problem** - `portfolio-data.js` blocked when opened locally
4. **Tested multiple solutions** - Embedded data, standalone files, server requirements

### Key Decisions

### Technical Lessons
- **Browser CORS:** `file://` protocol blocks external file loading (security feature)
- **Online vs Local:** Different blocking mechanisms - local=CORS, online=missing files
- **Iframe limitations:** Cannot load local files in iframes
- **Data architecture:** Embedded data works everywhere, external files need server
